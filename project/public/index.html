Welcome to Heart Track

Your IoT solution for monitoring heart rate and blood oxygen saturation.

Introduction

The Heart Track application is an IoT-enabled web app for monitoring heart rate and blood oxygen saturation levels throughout the day. The system uses a custom IoT device with a heart rate and oxygen sensor, which periodically reminds users to take measurements.

  • Data is transmitted from the IoT device to our secure web application.
  • Users can view their historical data through daily and weekly summaries.
  • The time of day and rate at which measurements are taken is fully configurable by the user.
  • The web application features a responsive design, allowing seamless viewing on desktops, tablets, and mobile devices.
